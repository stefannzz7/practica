HyperText Markup Language (HTML) este un limbaj de marcare utilizat pentru crearea paginilor web ce pot fi afișate într-un browser (sau navigator). Scopul HTML este mai degrabă prezentarea informațiilor – paragrafe, fonturi, tabele ș.a.m.d.

HTML

JavaScript (JS) este un limbaj de programare orientat obiect bazat pe conceptul prototipurilor. Este folosit mai ales pentru introducerea unor funcționalități în paginile web, codul JavaScript din aceste pagini fiind rulat de către browser.

Javascript

CSS (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML. Stilurile se pot atașa elementelor HTML prin intermediul unor fișiere externe sau în cadrul documentului

CSS

Angular este o platformă de dezvoltare web cu sursă deschisă bazată pe limbajul TypeScript. Proiectul este dezvoltat de Echipa Angular de la Google și de o comunitate de utilizatori individuali și companii. Angular este o rescriere completă, de către aceeași echipă, a frameworkului AngularJS.

Angular

Numele PHP provine din limba engleză și este un acronim recursiv : Php: Hypertext Preprocessor. Folosit inițial pentru a produce pagini web dinamice, este folosit pe scară largă în dezvoltarea paginilor și aplicațiilor web. Se folosește în principal înglobat în codul HTML, dar începând de la versiunea 4.3.0 se poate folosi și în mod „linie de comandă” (CLI), permițând crearea de aplicații independente.

PHP

Bootstrap este un cadru CSS liber și open-source, orientat către dezvoltarea web front-end receptivă și mobilă. Conține șabloane de design bazate pe CSS și JavaScript (opțional) pentru tipografie, formulare, butoane, navigare și alte componente de interfață.

Bootstrap

React (cunoscut și ca React.js sau ReactJS) este o bibliotecă JavaScript open-source pentru construirea interfețelor utilizatorului. Este întreținut de Facebook și de o comunitate de dezvoltatori și companii individuale

REACT

Vue.js este un cadru JavaScript de open-source-viewmodel pentru modelul open source pentru construirea de interfețe de utilizator și aplicații cu o singură pagină. A fost creat de Evan You și este întreținut de el și de restul membrilor activi ai echipei de bază, veniți de la diverse companii precum Netlify și Netguru.

VUE

Stackoverflow este platforma atotștiutoare de cod, paleta noastră pentru gândaci și umărul pe care putem plânge când ne dă eroare compilatorul.
